HTML Cheat Sheet — Complete Tag Reference

This page is a comprehensive HTML reference that demonstrates every major tag with explanations and examples. It’s written using pure HTML, without any CSS or JavaScript.

Basic Structure

<!DOCTYPE html> Declares the document type and version of HTML (HTML5). It must be the first line in an HTML document.
<html> </html> The root element that wraps all the content of the page. It represents the start and end of the HTML document.
<head> </head> Contains metadata and information about the document — not visible to users (e.g., title, meta, links).
<title> </title> Defines the title of the page that appears in the browser tab and search results.
<meta> Provides metadata such as description, author, and character encoding.
<link> Connects the document with external resources like a favicon or stylesheet.
<body> </body> Contains all the visible content of the page, including text, links, images, and sections.

<!-- --> or <! Comments at a point or across a line.

Example:


        <!DOCTYPE html>
        <html>
        <head>
        <title>My example</title>
        <meta charset="UTF-8"> <!--Define the character set used-->
        <link rel="stylesheet" href="mystyle.css"> <!--Link to external style sheets-->
        </head>
        <body>
        </body>
        </html>

Results:

All that is currently displayed is the title in the browser tab

Text and Headings

<h1>...<h6> Define headings, from most important (<h1>) to least (<h6>).
<p></p> Defines a paragraph of text.
<br> Inserts a line break.
<hr> Inserts a horizontal line separating content.
<b></b> Makes text bold (without extra meaning).
<strong></strong> Defines important text — usually displayed bold.
<i></i> Displays text in italics (for stylistic purposes).
<em></em> Emphasizes text — usually italic, but semantically meaningful.
<u></u> Underlines text.
<mark></mark> Highlights text with a background color.
<small></small> Makes text smaller.
<pre></pre> Displays preformatted text, preserving spaces and line breaks.
<blockquote></blockquote> Indicates a long quotation block.
<q></q> Indicates a short inline quote.
<abbr></abbr> Defines an abbreviation, with an optional title for full form.
<cite></cite> Specifies the title of a creative work (book, film, etc.).
<dfn></dfn> Marks a term being defined.

Lists and Text Grouping

<ul> Defines an unordered list (bulleted list).
<ol> Defines an ordered list (numbered list).
<li> Defines a list item inside <ul> or <ol>.
<div> Generic block container for grouping content.
<span> Generic inline container for text.
<address> Represents contact information for author or organization.
<section> Defines a section of content, typically with a heading.
<article> Defines an independent, self-contained content piece.
<header> Represents introductory content or a group of navigational links.
<footer> Defines a footer for a section or page.
<main> Specifies the main content of a document.

Example:


        <ul>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
        </ul>

        <ol>
        <li>Step one</li>
        <li>Step two</li>
        <li>Step three</li>
        </ol>

        <div>
        <header><h2>Header of a section</h2></header>
        <article>
            <p>This is an article inside a div.</p>
            <span>Inline span text</span>
        </article>
        <footer>Footer text</footer>
        </div>

        <main>
        <section>
            <h3>Contact</h3>
            <address>Author: Panagiotis Stivaktas</address>
        </section>
        </main>
