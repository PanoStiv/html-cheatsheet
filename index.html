HTML Cheat Sheet — Complete Tag Reference

This page is a comprehensive HTML reference that demonstrates every major tag with explanations and examples. It’s written using pure HTML, without any CSS or JavaScript.

Basic Structure

<!DOCTYPE html> Declares the document type and version of HTML (HTML5). It must be the first line in an HTML document.
<html> </html> The root element that wraps all the content of the page. It represents the start and end of the HTML document.
<head> </head> Contains metadata and information about the document — not visible to users (e.g., title, meta, links).
<title> Defines the title of the page that appears in the browser tab and search results.
<meta> Provides metadata such as description, author, and character encoding.
<link> Connects the document with external resources like a favicon or stylesheet.
<body> </body> Contains all the visible content of the page, including text, links, images, and sections.

<!-- --> or <! Comments at a point or across a line.

Example:


    <!DOCTYPE html>
    <html>
    <head>
      <title>My example</title>
      <meta charset="UTF-8"> <!--Define the character set used-->
      <link rel="stylesheet" href="mystyle.css"> <!--Link to external style sheets-->
    </head>
    <body>
    </body>
    </html>
